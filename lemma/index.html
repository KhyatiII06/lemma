    How VoiceCare AI Works

    From dialect voice input to Hindi voice response and ASHA dispatch — in under 3 seconds.

    🎤
    Input Layer
    Patient speaks in any dialect
    The user sends a WhatsApp voice note or uses the mic in their local language — Nagpuri, Santali, Bhojpuri, Chhattisgarhi, Maithili, Hindi. No typing, no app, no literacy required.
    Web Speech APIMediaRecorderSpeechRecognition
    🧠
    AI Layer
    Narrative NLP extracts symptoms
    Claude AI analyses the unstructured story, not a form. It understands context: "mere pet mein 3 din se aag hai" → stomach burning for 3 days. No rigid Q&A. A 97-keyword dataset provides instant offline fallback.
    Claude claude-sonnet-4-6Keyword DatasetFastAPI
    ⚖️
    Triage Layer
    Urgency scored: mild / moderate / critical
    Dual-path scoring: local keyword scan (instant, offline) + Claude AI semantic analysis. Claude overrides keyword hints when context suggests higher risk. Result is one of three urgency levels with specific response routing.
    Local Keyword EngineClaude APIPydantic
    🔊
    Output Layer
    Hindi voice message sent back
    The AI response is converted to audio using the browser's native Hindi Text-to-Speech (Web Speech API, hi-IN). Female voice selected — research shows it's more trusted in rural communities. Zero text displayed. Pure audio.
    Web Speech API (TTS)hi-IN VoiceSpeechSynthesis
    🚨
    Emergency Layer
    ASHA worker SOS + GPS dispatch
    If critical: a parallel thread fires immediately — ASHA worker gets SMS + WhatsApp alert with patient GPS, symptom summary, and PHC contact. This happens while the patient is still hearing the calm Hindi voice message. Zero delay.
    Twilio SMSWhatsApp APIGPS Location
    📡

    Lemma SDK — Production Monitoring for Every Triage Call

    Every triage spans from input to output is traced using the Lemma OpenTelemetry SDK. Lemma detects when urgency classifications diverge from expected patterns, surfaces failure clusters (e.g. "chest pain in Chhattisgarhi consistently classified as moderate"), and suggests prompt improvements. This closes the loop between deployment and accuracy — VoiceCare gets smarter with every real-world use.
